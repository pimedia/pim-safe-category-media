WordPress plugin that adds a field on the WP Admin Category page.

== Description ==

WordPress plugin that adds a field on the WP Admin Category page that allows users to upload or select an existing image in either PNG or JPEG format from the WordPress Media Library. Features include safe deletion of media, prevents users from deleting any images from the WordPress Media Library if the image is being used in posts.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `safe-category-media` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php do_action('plugin_name_hook'); ?>` in your templates
